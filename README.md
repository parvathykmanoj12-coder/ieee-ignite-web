# IEEE Ignite 2026 Website

An immersive, high-performance, and fully responsive technical convergence web platform designed and engineered for the IEEE Student Branch event presentation.

## 🚀 Live Visuals & Links
* **Live Website Link:** https://ignite-ieee-web.netlify.app
* **GitHub Repository Link:** https://github.com/parvathykmanoj12-coder/ieee-ignite-web

## ✨ Key Features Implemented

* **💻 Premium UI Design** | Ultra-modern cybernetic aesthetic built on a clean light theme baseline with custom typography scales.
* **✨ Canvas Particle Engine** | Native HTML5 interactive background node matrix tuned to a smooth, elegant velocity layout.
* **🔍 Multi-Category Filter** | Smart event sorting by category slices (`Technical`, `Professional`, `Interactive`) and instant text search.
* **📍 Scroll-Spy Tracking** | Fluid viewport intersection tracking to dynamically highlight active links as you scroll.
* **📱 Desktop-to-Mobile Rows** | Pure CSS Flexbox engineering that keeps navigation completely inline and responsive across all screens.

## 🚀 Developed With
* Semantic HTML5 Markup Architecture
* Custom CSS3 Property Modules (Variables, Layout Gradients)
* Vanilla JavaScript (ES6+ Asynchronous Event Frameworks)
